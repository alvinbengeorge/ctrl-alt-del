# ctrl-alt-del
Project for Quantathon

# `Installation`

```sh
pip3 install -r requirements.txt
cd frontend
npm install
```

# `Backend`

```sh
# Deployment
uvicorn main:app

# Development
uvicorn main:app --reload
```

# `FrontEnd`

```sh
# Deployment
npm run build
npm start

# Development
npm run dev
```
